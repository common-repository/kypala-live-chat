== Description ==

Kypala Live Chat allows you to chat with visitors to your site via a customizable chat window.  You can customize the chat window to match your site by changing colors, fonts, and other details.

== Installation ==

1.  Go to www.kypala.com and start an account.
2.  Activate the plugin through the 'Plugins' screen in WordPress


== Frequently Asked Questions ==

= Do I need an account at kypala.com? =

Yes, to use the chat, first go to kypala.com and start an account.
